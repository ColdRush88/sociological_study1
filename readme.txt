Дополнительные данные для работы "Уровень поддержки специальной военной операции на территории Украины гражданами Российской Федерации. Социологический опрос."
Описание файлов
socStudy1_rawData.txt - ответы респондентов; каждая строка соответствует одному дню (см. Таблицу 1 в статье)
socStudy1_analysis.cpp - исходный код программы использованной для оценки доверительных интервалов
socStudy1_analysis.out - программа использованная для оценки доверительных интервалов (скомпилирована на Debian10, g++)
compilationCommand.txt - строка для компиляции socStudy1_analysis.cpp (должна быть установлена библиотека GSL)

Supplementary data for the work "The level of support of the special military operation on the territory of Ukraine among citizens of the Russian Federation". The sociological survey."
File description
socStudy1_rawData.txt - answers of respondents; each line corrsponds to one day (see Table 1 in the article)
socStudy1_analysis.cpp - source code of the program for the evaluation of confidence intervals
socStudy1_analysis.out - the program for the evaluation of confidence intervals (compiled under Debian10, g++)
compilationCommand.txt - the command for the compilation of socStudy1_analysis.cpp (depends upon GSL library)
